# 🌤️ 城市天气查询

一个简洁美观的天气查询 Web 应用，支持全球城市天气查询。

## 功能特点

- 🌍 支持全球城市天气查询
- 🎨 现代化渐变设计风格
- 📱 响应式设计，支持移动端
- ⚡ 实时天气数据
- 🌡️ 显示温度、湿度、风速等详细信息
- 🔥 无需 API Key，完全免费

## 技术栈

- HTML5
- CSS3 (Flexbox + Grid)
- Vanilla JavaScript (ES6+)
- wttr.in API (免费天气数据)

## 在线预览

直接在浏览器中打开 `index.html` 即可使用。

## 本地运行

```bash
# 克隆项目
git clone https://github.com/noobpeter/weather-app.git

# 进入目录
cd weather-app

# 用浏览器打开
open index.html
```

## 使用说明

1. 在输入框中输入城市名称（如：北京、Shanghai、London、New York）
2. 点击"查询"按钮或按回车键
3. 查看天气信息

## API 说明

本项目使用 [wttr.in](https://wttr.in) 提供的免费天气 API，无需注册即可使用。
